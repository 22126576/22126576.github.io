Technology Systems

Understanding Core IT Infrastructure Content:

This unit covers fundamental components of IT systems, such as computer hardware, software, and networks. It explores how different technologies work together to create functional systems, how data is processed, and how users interact with these systems. Additionally, students will learn about maintaining and troubleshooting technology systems to ensure smooth operation.

Assessment Overview

Assessment is via on online test after covering all the different topics in this unit to evaluate our grade. This test was mostly multiple choice with a writing task at the end.

Topics Covered in this unit:

  • Technology systems in different sectors
  • Computing Devices
  • Input and Output Devices
  • Storage Devices
  • Automated Technology Systems
  • Types of Computer Networks
  • Developments in Technology
  • Issues with Technology Systems
  • Uses of Technology Systems
  • Data Transfer methods
  • Internal Components
  • Computer Performance Factors
